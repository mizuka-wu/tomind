# @tomind/extensions

内置扩展集合：Keymap、Viewport、Selection、Relationship 等 20+ 扩展。

## 导出

- `KeymapExtension` — 快捷键管理
- `ViewportExtension` — 视口控制
- `SelectDragExtension` — 选区拖拽
- `MouseBoxSelectExtension` — 鼠标框选
- `DropExtension` — 拖放处理
- `ContextMenuExtension` — 右键菜单
- `ResizeBoxExtension` — 缩放框
- `RelationshipExtension` — 关联线交互
- `EditBridgeExtension` — 双击编辑桥接
- `ThemeExporterExtension` — 主题导出
- `MatrixExtension` — 矩阵布局
- 等 20+ 扩展
