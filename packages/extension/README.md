# @tomind/extension

Tiptap 风格 Extension 系统：创建、配置、管理扩展。

## 导出

- `createExtension()` / `createNodeExtension()` / `createPartExtension()` — 创建扩展
- `ExtensionManager` — 扩展管理器（setup, register, unregister）
- `Extension` / `ExtensionContext` / `CommandFn` — 类型定义

## 依赖

- `@tomind/schema`
- `@tomind/state`
